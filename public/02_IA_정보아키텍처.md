## 4. 데이터 흐름 (Data Flow)

### 4.1 학습 사이클

```
[사용자 진입]
    │
    ▼
[진단평가] ─── 과목별 20문항 ──→ [현재 수준 파악]
    │                                    │
    ▼                                    ▼
[맞춤 학습계획 생성] ◄───── [취약영역 분석]
    │
    ├── Phase 1: 개념카드 학습 (에빙하우스 반복)
    │       │
    │       ▼
    ├── Phase 2: 문제풀이 (난이도 적응형)
    │       │
    │       ▼
    ├── Phase 3: 모의고사 (실전 시뮬레이션)
    │       │
    │       ▼
    └── Phase 4: 취약점 보강 ──→ [반복 사이클]
```

### 4.2 적응형 난이도 조절

```
[문제 출제]
    │
    ├── 정답률 80%↑ ──→ 난이도 상향 (기초→응용→실전)
    │
    ├── 정답률 50~80% ──→ 현재 난이도 유지
    │
    └── 정답률 50%↓ ──→ 난이도 하향 + 개념카드 복습 추천
```
