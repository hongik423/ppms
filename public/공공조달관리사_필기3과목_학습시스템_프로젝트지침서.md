## 1. 프로젝트 개요

### 1.1 프로젝트 목적

본 프로젝트는 2026년 최초 시행되는 **공공조달관리사** 국가기술자격 **필기시험 제3과목**에 대해, 표준교재 제3권 「공공계약관리」 전체 내용과 출제기준 4개 주요항목을 교차분석하여 **출제 가능성이 가장 높은 핵심 영역을 단시간에 집중 학습**할 수 있는 AI 기반 학습 시스템을 개발하는 것을 목적으로 한다.

#### 📝 핵심 정리

**[3.1.1] 계약관리 원칙 ⭐⭐**

**가. 계약 자유의 4가지 원칙:**

| 원칙 | 핵심 내용 |
|---|---|
| **체결의 자유** | 계약을 체결할지 여부를 자유롭게 결정 |
| **상대방 선택의 자유** | 계약 상대방을 자유롭게 선택 |
| **내용결정의 자유** | 계약 내용(조건)을 자유롭게 결정 |
| **방식의 자유** | 계약 형식을 자유롭게 결정 |

> **공공계약 특수성**: 국가계약법 제5조에 의해 계약 자유가 **제한**됨 (법정 절차 의무, 경쟁입찰 원칙 등)

**나. 계약의 기능 5가지 ⭐:**

| 기능 | 핵심 내용 |
|---|---|
| ① 재화의 이전 | 물품·용역의 소유권 또는 이용권 이전 |
| ② 재화의 교환 | 대금과 재화의 상호 교환 |
| ③ 신용의 수여 | 선급금, 기성금 등 신용 제공 |
| ④ 위험의 배분 | 이행불능, 물가변동 등 위험을 당사자 간 배분 |
| ⑤ 분쟁의 해결 | 계약 조건을 통한 분쟁 예방 및 해결 기준 마련 |

**다. 계약 성립요건 4가지 ⭐⭐:**

| 요건 | 핵심 내용 |
|---|---|
| **당사자** | 계약 능력이 있는 당사자(발주기관 + 계약상대자) |
| **의사표시의 합치** | 청약과 승낙의 합치 |
| **확정 가능한 내용** | 계약 목적물, 대가 등이 확정 또는 확정 가능 |
| **적법한 내용** | 법률에 위반되지 않는 내용 |

**라. 계약서 해석 순위 4원칙 ⭐:**

| 순위 | 해석 기준 | 핵심 내용 |
|---|---|---|
| 1순위 | **계약 특수조건(SC)** | 해당 계약에만 적용되는 특별 조건 |
| 2순위 | **계약 일반조건(GC)** | 모든 계약에 공통 적용되는 일반 조건 |
| 3순위 | **설계서/과업지시서** | 기술적 내용 및 사양 |
| 4순위 | **산출내역서** | 비용 산출 근거 |

**[3.1.2] 계약변경 관리 ⭐⭐⭐ (출제 가능성 매우 높음)**

**가. 변경계약 절차 6단계 ⭐⭐:**

| 단계 | 핵심 내용 |
|---|---|
| **1단계: 변경사유 발생** | 설계변경, 물가변동, 공기연장, 기타 불가항력 |
| **2단계: 변경요청** | 계약상대자 또는 발주기관이 변경요청서 제출 |
| **3단계: 변경검토** | 변경 사유의 적정성, 금액 조정의 타당성 검토 |
| **4단계: 변경승인** | 발주기관 결재권자의 변경 승인 |
| **5단계: 변경계약 체결** | 변경계약서 작성·서명 |
| **6단계: 이행관리** | 변경된 내용에 따른 이행 관리 |

**나. 물가변동 계약금액 조정 ⭐⭐⭐:**

| 항목 | 핵심 내용 |
|---|---|
| **조정 요건** | 계약 체결 후 **90일** 이상 경과 + 조정률 **100분의 3** 이상 증감 |
| **조정 방법** | ①**품목조정률** 방식 ②**지수조정률** 방식 (택1) |
| **품목조정률** | 개별 품목(자재·노무·경비)의 등락률을 반영하여 계산 |
| **지수조정률** | 한국은행 발표 물가지수 등을 기준으로 일괄 산정 |
| **조정 시기** | 90일 경과 후 조정률 3% 이상 충족 시점 |
| **재조정** | 최초 조정 후 다시 90일 경과 + 3% 이상 변동 시 재조정 가능 |

**다. 설계변경 사유 4가지 ⭐⭐:**

| 사유 | 핵심 내용 |
|---|---|
| **①설계의 오류** | 설계서의 내용이 불분명하거나 누락된 경우 |
| **②현장 여건 상이** | 지질, 용수 등 현장 조건이 설계와 다른 경우 |
| **③신기술 적용** | 새로운 기술·공법 적용이 필요한 경우 |
| **④발주기관 요구** | 발주기관의 필요에 의한 설계 변경 |

**라. 손해배상 절차 및 사후조치 ⭐⭐:**

| 항목 | 핵심 내용 |
|---|---|
| **손해배상 사유** | 계약상대자의 귀책사유로 발주기관에 손해 발생 시 |
| **배상 범위** | 실제 발생한 손해 + 일실이익(간접손해는 특약 시) |
| **절차** | 손해 발생 → 손해액 산정 → 배상 청구 → 협의 → 지급 또는 소송 |
| **보증금 활용** | 계약보증금에서 우선 충당 가능 |
| **관련 법조문** | 민법 제390조(채무불이행), 제551조(해제·해지와 손해배상 병존) |

**마. 분쟁해결 절차 ⭐⭐:**

| 방법 | 핵심 내용 |
|---|---|
| **당사자 협의** | 계약 당사자 간 직접 협의를 통한 해결 (1차) |
| **이의신청** | 발주기관에 15일 이내 이의신청 → 15일 이내 결과 통보 |
| **국가계약분쟁조정** | 분쟁조정위원회 조정 신청 → 50일 이내 심사·조정 |
| **중재** | KCAB(대한상사중재원) 등을 통한 중재 → 확정판결과 동일 효력 |
| **행정소송/민사소송** | 최종적 법적 구제수단 |

**바. 계약해제·해지 절차 ⭐⭐:**

| 구분 | 해제 | 해지 |
|---|---|---|
| **효과** | **소급적** (처음부터 무효) | **불소급적** (장래에 대해 소멸) |
| **원상회복** | 있음 | 없음 |
| **사유** | 부정당행위, 중대한 계약위반 | 이행지체, 이행불능, 파산 등 |
| **절차** | 사유 발생 → 최고 → 해제 통지 → 보증금 귀속 → 정산 | 사유 발생 → 최고 → 해지 통지 → 기성부분 정산 |

**[3.1.3] 계약이행 관리 ⭐⭐**

**가. 계약체결 후 착수협의 및 이행계획 ⭐⭐:**

| 항목 | 핵심 내용 |
|---|---|
| **착수협의** | 계약체결 후 발주처와 계약상대자 간 착수시점, 보고일정, 납품대상 등 구체 협의 |
| **착수시점** | 계약서에 명시된 착수일 또는 착수협의에서 합의한 일자 |
| **보고(점검)일정** | 중간보고, 정기점검, 수시점검 일정 확정 |
| **투입인력 계획** | 사업관리자, 기술인력 등 투입인력 및 시기 확정 |
| **소요자원 계획** | 장비, 재료, 시설 등 계약이행에 필요한 자원 투입 계획 |

**나. 계약이행 일반 절차:**
1. 착수계(착수신고) 제출 → 2. 착수협의 → 3. 이행 관리(감독) → 4. 중간 보고/점검 → 5. 납품/완료 → 6. 검사·검수 → 7. 대금 지급

**다. 공급 계약조건 이행점검 ⭐:**

| 점검 항목 | 핵심 내용 |
|---|---|
| **납기 준수** | 계약서 명시 납품기한 이행 여부 |
| **품질 기준** | 규격서·과업지시서 기준 충족 여부 |
| **투입인력** | 계약서 명시 인력 투입 여부 |
| **하도급 적정성** | 승인된 하도급 조건 이행 여부 |
| **미이행·오류·지연 관리** | 이행 미달 시 시정요구 → 보완 → 재시정 → 계약해제·해지 |

**나. 계약성과관리(SLA/PBSC) ⭐:**

| 구분 | SLA | PBSC |
|---|---|---|
| **명칭** | Service Level Agreement | Performance-Based Service Contracting |
| **정의** | 서비스 수준 합의서 | 성과기반 서비스계약 |
| **적용** | IT서비스, 시설관리 등 | 성과 중심 용역 |
| **핵심** | 서비스 품질 지표 설정 및 이행 | 투입이 아닌 결과(성과) 기준 평가 |

**다. 계약상 위험 관리:**
- **위험 유형**: 이행지체, 품질 불량, 불가항력, 공급중단, 재무위험 등
- **대응 방안**: 계약보증금, 이행보증보험, 하자보수보증금, 지체상금 등

**[3.1.4] 계약종결 관리 ⭐⭐⭐ (출제 가능성 매우 높음)**

**가. 납품검사·검수 ⭐⭐:**

| 항목 | 핵심 내용 |
|---|---|
| **검사 기한** | 납품 또는 이행 완료 후 **14일** 이내 |

**다. 원가계산 구성체계 ⭐⭐⭐ (출제 가능성 매우 높음):**

```
직접공사비 = 직접재료비 + 직접노무비 + 직접경비
    +
간접공사비 = 간접노무비 + 현장경비
    ↓
순공사원가
    +
일반관리비
    +
이윤
    ↓
총원가
    +
부가가치세(VAT)
    ↓
건설공사금액
```

| 구성 요소 | 산출 기준 | 핵심 내용 |
|---|---|---|
| **직접재료비** | 실적 단가 | 공사에 직접 투입되는 재료비 |
| **직접노무비** | 노임단가 × 공수 | 공사에 직접 참여하는 근로자 인건비 |
| **직접경비** | 실비 | 가설비, 운반비, 기계경비 등 |
| **간접노무비** | 직접노무비 × 비율 | 현장관리자, 보조인력 인건비 |
| **현장경비** | 실비 또는 비율 | 현장사무실, 안전관리비 등 |
| **일반관리비** | (직접재료비+노무비+경비) × 비율 | 본사 경비, 관리비 |
| **이윤** | (노무비+경비+일반관리비) × 비율 | 적정 이윤 (15% 이내) |
