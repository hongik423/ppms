## 3. 주요항목별 핵심 내용 심층분석

### 3.1 [주요항목1] 공공조달 개요 (예상 3문항, 10%)

#### 📌 출제기준 세세항목
1. 공공조달의 정의, 목적, 범위
2. 공공조달의 참여자와 이해관계자
3. 공공조달과 민간조달(구매)의 특성과 차이
4. 공공조달 프로세스(순환적/연속적)
5. 공공조달 구성체계(조직적/법적/운영적)에 따른 실행 및 관리

#### 📝 핵심 정리

**[3.1.1] 공공조달의 정의**

- **공공(Public)**: 정부, 지방자치단체, 공공기관 등 공공부문
- **조달(Procurement)**: 구매, 획득, 공급
- **정의**: 정부나 지자체 등 공공부문에서 필요한 물품이나 용역(서비스), 공사 등을 구매하는 절차
- 공공재정을 활용해 국민의 세금으로 한 물품, 용역, 공사를 구매하는 일련의 연계된 활동
- **핵심 원칙**: VFM(Value For Money, 비용 대비 가치), 윤리(Ethics), 경쟁(Competition), 투명성(Transparency), 책임성(Accountability)
- **규모**: 전 세계 공공조달 규모 = 국가 GDP의 **12~15%**, OECD 회원국 정부 총지출 중 조달지출 비중 약 **34%** (2023년 기준)
- 한국: GDP 대비 공공조달 지출 비중 약 **11%**, 정부 총지출 대비 약 **34.3%**

**[3.1.2] 공공조달의 중요성 (4가지 측면)**

| 측면 | 핵심 내용 |
|---|---|
| ① 재정 운영 효율성 | 정부 지출의 약 1/3 차지 → 효율적 운영이 비용 대비 가치 확보의 핵심 |
| ② 국민 삶의 질 | 건강, 환경, 인프라, 운송, 에너지, R&D 등 모든 공공서비스 연계 |
| ③ 고위험 영역 | 복잡한 가치사슬, 부패/청렴성 위험, IT/재무/평판/환경 리스크 증가 |
| ④ 전략적 활용 | 지속가능성, 혁신, 사회적 책임, 중소기업 장려 등 정책 목표 달성 수단 |

**[3.1.3] 공공조달 vs 민간조달 차이 (출제 포인트)**

| 구분 | 공공조달 | 민간조달 |
|---|---|---|
| 재원 | 국민 세금(공공재정) | 기업 자체 자금 |
| 규정 | 법령·계약예규 엄격 적용 | 내부 규정 자율 적용 |
| 목적 | 공공서비스 실행 + 정책 목표 | 이윤 극대화 |
| 가치 | 사회적 가치, 환경, ESG 고려 필수 | 선택적 고려 |
| 투명성 | 법적 의무 (입찰공고, 결과공개) | 자율적 결정 |

**[3.1.4] 공공조달의 발달과정**

| 시대 | 조달대상 | 실행방법 | 법적체계 | 조직적 체계 |
|---|---|---|---|---|
| 고대 | 대규모 건축/토목 | 국가 직접 통제, 노동력 직접 조달 | 공공조달 법적 체계 부재 | 사업별 담당 조직 |
| 중세 | 궁궐/도로/교량 | 왕실·영주에게 노동력 대금 지급 | 초기 법적 체계 출현 | 왕실 특허상인 독점적 공급 |
| 근대 | 산업혁명 자재, 군수물자 | 대규모 입찰 방식 적용 | 국가적 수준 계약법령 출현 | 전문적 중앙조달 기관/부제 출현 |
| 현대 | 공공서비스 전 분야 | 입찰·경쟁 공식적 조달 실행 | 국제적 법적 프레임워크 | 중앙집중화 기반 하이브리드 운영 |

- **한국 조달청**: 1961년 경제개발계획 이후 설치, 공공조달 기능과 역할 공식화
- **KONEPS(나라장터)**: 2002년 서비스 개통, 유엔 전자정부 대상 수상, 세계적 모범 사례

**[3.1.5] 공공조달 구성체계 (3대 체계)**

| 체계 | 핵심 내용 |
|---|---|
| **법적 체계** | 국가계약법, 지방계약법, 조달사업법, 전자조달법 등 법률·시행령·시행규칙 |
| **조직적 체계** | 중앙조달기관(조달청), 수요기관, 공공기관, 지자체 |
| **운영적 체계** | 중앙집중형, 분산형, 하이브리드형 (한국은 하이브리드) |

### 3.2 [주요항목2] 공공조달 원칙 및 방법 (예상 4~5문항, 15%)

#### 📌 출제기준 세세항목
1. 공공조달의 기본 원칙 (5대 원칙)
2. 공공조달의 실행 원칙 (OECD 12대 원칙)
3. 공공조달의 경쟁적 방법 (일반·제한·지명경쟁)
4. 공공조달의 비경쟁적 방법 (수의계약)
5. 중앙조달(구매)기관의 역할과 활용 (FA, DPS, MAS, e-Catalogue)

#### 📝 핵심 정리

**[3.2.1] 5대 기본원칙 ⭐ (반드시 암기)**

| 원칙 | 영문 | 핵심 내용 |
|---|---|---|
| **투명성** | Transparency | 조달 과정의 모든 정보를 공개하고 접근 가능하게 |
| **비용 대비 가치** | Value for Money (VFM) | 단순 최저가가 아닌 총생애비용 기반 최적 가치 확보 |
| **경쟁** | Competition | 공정하고 개방적인 경쟁을 통한 최적 공급자 선정 |
| **차별금지/동등대우** | Non-discrimination | 모든 공급자에게 공평한 기회와 동등한 조건 보장 |
| **책임성** | Accountability | 조달 관련 의사결정과 결과에 대한 명확한 책임 부여 |

**[3.2.2] OECD 12대 실행원칙**

| 순번 | 원칙 | 핵심 키워드 |
|---|---|---|
| 1 | 투명성 | 이해관계자에게 동등한 정보 공개 |
| 2 | 청렴성 | 부패 방지, 이해충돌 관리 |
| 3 | 접근성 | 중소기업 등 모든 잠재 공급자 참여 보장 |
| 4 | 참여 | 이해관계자 참여를 통한 전문성 활용 |
| 5 | 효율성 | 5R원칙(적정 수량·품질·시기·가격·장소) |
| 6 | 균형성 | 정책 목표 간 균형 유지 |
| 7 | 전자조달시스템 활용 | End-to-End 전자화 |
| 8 | 역량 개발 | 조달 전문인력 육성 |
| 9 | 통합 | 조달 정책과 기타 정부 정책 간 연계 |
| 10 | 평가 | 조달 성과의 체계적 측정·모니터링 |
| 11 | 위험관리 | 조달 전 과정의 리스크 식별·대응 |
| 12 | 책임성 | 내부 통제와 외부 감사 체계 운영 |

> **효율성의 5R원칙**: Right Quantity(적정 수량), Right Quality(적정 품질), Right Time(적정 시기), Right Price(적정 가격), Right Place(적정 장소)

**[3.2.3] 경쟁적 조달방법 비교 ⭐**

| 방법 | 요건 | 적용 상황 |
|---|---|---|
| **일반경쟁** | 자격요건 충족 모든 업체 참가 가능 | 원칙적 방법 (국가계약법 기본) |
| **제한경쟁** | 실적·기술·자본금 등으로 참가 제한 | 특수 전문성 필요 시 |
| **지명경쟁** | 수요기관이 특정 업체를 지명 | 긴급·비밀·특수 기술 필요 시 |

**[3.2.4] 비경쟁적 방법: 수의계약 ⭐**

- 경쟁 입찰 없이 특정 공급자와 직접 계약 체결
- **허용 사유**: 천재지변·비상사태, 특허·전매, 추정가격 이하 소액, 국방·안보 관련 등
- **1인 수의계약**: 특정 사유에 한해 1개 업체와만 계약
- **2인 이상 견적 수의계약**: 2개 이상 업체 견적 비교

**[3.2.5] 중앙조달기관 활용 방법**

| 방법 | 약칭 | 핵심 내용 |
|---|---|---|
| **기본협약** | FA (Framework Agreement) | 일정기간 동안 물품/서비스 공급 기본 조건 사전 합의 |
| **동적구매시스템** | DPS (Dynamic Purchasing System) | 표준화된 반복 구매 지원, 공급업체 수시 가입 가능 |
| **다수공급자계약** | MAS (Multiple Award Schedule) | 동일 품목 2인 이상 계약, 수요기관 선택 가능 |
| **전자카탈로그** | e-Catalogue | 표준화된 상품 목록을 통한 간편 구매 |

> **FA vs DPS 차이**: FA는 사전 검증 완료 공급업체 대상으로 일정기간 기본계약, DPS는 공급업체가 언제든지 가입 가능한 실시간 온라인 시스템

### 3.3 [주요항목3] 전자조달시스템 (예상 4~5문항, 15%)

#### 📌 출제기준 세세항목
1. 전자조달시스템의 도입 배경 및 발전과정
2. 전자조달시스템 기능 간 연계성
3. 전자조달시스템 기능과 기대효과
4. 나라장터를 통한 전자입찰, 전자계약, 전자지불 절차
5. 나라장터와 연계한 계약지원 플랫폼
6. 전자입찰을 위한 가격유형, 세부처리 절차
7. 물품목록제도, 목록화, 목록정보시스템

#### 📝 핵심 정리

**[3.3.1] 전자조달시스템 개요**

- **정의**: 인터넷 등 정보통신 네트워크를 기반으로 상품이나 용역의 소싱, 구매, 지불 등을 자동화·전자화한 시스템
- **KONEPS(Korea ON-line Electronic Procurement System)**: 국가종합전자조달시스템 = 나라장터
- 2002년 서비스 개통, 유엔 전자정부 대상 수상, 세계적 모범 사례
- 한국은 1997년부터 조달 절차 간소화 → 2000년 11월 전자입찰시스템 구축 → 2002년 KONEPS 운용 시작

**[3.3.2] 전자조달시스템 유형 (2가지 범주) ⭐**

| 유형 | 핵심 기능 | 주요 도구 |
|---|---|---|
| **전자적 소싱(e-Sourcing)** | 공급자 탐색·평가, 입찰·계약 체결 | e-Tendering, e-Evaluation, e-Collaboration, e-Negotiation, e-Auction |
| **전자적 구매(e-Purchasing)** | 구매 실행, 지불 처리 | Purchasing, Payment, Purchasing Card, e-Invoicing |

**[3.3.3] 전자조달시스템 운영환경 - 4개 지출유형 분류 ⭐**

| 지출유형 | 복잡성/위험 | 지출규모 | 적합 전자조달 도구 |
|---|---|---|---|
| **기술적(Technical)** | 높은 복잡성, 낮은 지출 | 소규모 | e-Evaluation, e-Collaboration, 계약관리 중심 |
| **전략적(Strategic)** | 높은 복잡성, 높은 지출 | 대규모 | e-Tendering, EOI, 공개입찰 |
| **인수(Acquisition)** | 낮은 복잡성, 낮은 지출 | 소규모 | e-Catalogue, e-Payment, 워크플로 관리 |
| **레버리지(Leverage)** | 낮은 복잡성, 높은 지출 | 대규모 | e-Auction, e-Quoting, Framework Agreement |

**[3.3.4] 전자조달시스템 발전 5단계**

| 단계 | 내용 |
|---|---|
| 1단계 | 전통적 종이 기반 시스템 + 일부 컴퓨터 문서처리 |
| 2단계 | 메인프레임, PC, ERP, EDI 활용한 전자적 업무처리 |
| 3단계 | 이메일/웹사이트 등 인터넷 기반 커뮤니케이션 |
| 4단계 | 온라인 공급자 등록, 전자카탈로그, 웹 기반 EDI |
| 5단계 | 인터넷 기반 전자적 처리 도구와 전용 플랫폼으로 완전 대체 (현재) |

**[3.3.5] 전자조달시스템 핵심 구성요소 ⭐**

| 기능분류 | 핵심 요구사항 |
|---|---|
| **전자조달계획** | 상품/공사/서비스 예산 정의, 조달방법 설정, FA 수립 |
| **전자게시/알림** | 입찰공고 게시, 규격서/과업지시서 제공, 변경 공고 |
| **전자입찰 모듈** | 암호화된 입찰서 제출, 타임스탬핑, 접수 시간 관리 |
| **전자평가/낙찰** | 온라인 평가, 투찰서 사전열람 불가, 접근통제 |
| **전자계약관리** | 계약 결과물 세부정보 관리, KPI 모니터링, 재무 통합 |
| **전자구매** | 요청·견적·구매오더·입고·송장·지급 프로세스 지원 |
| **공급업체관리** | 입찰참가자격 관리, 적격심사, 실적 관리, 부적격 업체 참여 금지 |

**[3.3.6] 나라장터 연계 주요 플랫폼**

| 플랫폼 | 핵심 기능 |
|---|---|
| **종합쇼핑몰** | e-Catalogue 기반 상품 구매 (전자카탈로그) |
| **혁신장터** | 혁신제품 전용 구매 플랫폼 |

**[3.3.7] 전자입찰 보완도구 ⭐**

|---|---|

| **전자카탈로그(e-Catalogue)** | 표준화된 상품정보 등록, 조달기관이 온라인 비교·주문 |

| **기본협약(FA)** | 일정기간 기본 조건 사전 합의, 입찰절차 간소화 |

### 3.4 [주요항목4] 전략적 공공조달 (예상 4~5문항, 15%)

#### 📌 출제기준 세세항목
1. 중소기업지원 조달제도 (중소기업자간 경쟁제도 등)
2. 녹색조달(GPP) 및 지속가능 조달
3. 혁신조달 (혁신제품 지정, 혁신장터)
4. 사회적 가치 조달, ESG, 사회적경제기업 지원

#### 📝 핵심 정리

**[3.4.1] 전략적 공공조달(SPP) 개념 ⭐**

- **SPP(Strategic Public Procurement)**: 조달의 구매력을 활용해 경제·사회·환경적 정책 목표를 동시 달성하는 접근법
- 한국 전략적 공공조달 활용 비중: 평균 **62.23%** (4개 분야 합산)

| 분야 | 비중 | 핵심 제도 |
|---|---|---|
| **중소기업 지원** | 47.24% | 중소기업자간 경쟁제도, 직접생산확인제도 |
| **녹색조달(GPP)** | 5.87% | 녹색제품 우선구매, 환경표지인증, 저탄소제품 인증 |
| **혁신조달** | 3.28% | 혁신제품 지정, 혁신장터, 우수제품 지정 |
| **사회적 가치** | 9.96% | 사회적경제기업 우선구매, 장애인기업 우선구매, ESG |

**[3.4.2] 중소기업 지원 조달제도 ⭐⭐ (출제 가능성 높음)**

- **중소기업자간 경쟁제도**: 추정가격 일정 금액 이하 물품·용역·공사를 중소기업만 참가 가능하도록 제한
- **직접생산확인제도**: 중소기업이 직접 생산한 제품임을 확인하는 제도 (중소벤처기업부 확인서 발급)
- **중소기업제품 구매목표제**: 공공기관 물품 구매액의 일정 비율 이상을 중소기업 제품으로 구매 의무화
- **기술개발제품 우선구매**: 신기술(NET), 신제품(NEP), 녹색기술인증 제품 우선 구매

**[3.4.3] 녹색공공조달(GPP: Green Public Procurement) ⭐**

- **정의**: 공공기관이 상품·서비스를 구매할 때 환경영향을 고려하여 환경 친화적 제품을 우선 구매
- **법적 근거**: 녹색제품 구매촉진에 관한 법률
- **환경표지인증**: 환경부가 인증한 녹색제품
- **GPP 목표제**: 공공기관별 녹색제품 구매 목표율 설정·이행·공개

**[3.4.4] 혁신조달 ⭐**

- **혁신제품 지정제도**: 조달청이 혁신성이 우수한 제품을 지정 → 수의계약 또는 우선구매 가능
- **혁신장터**: 혁신제품 전용 구매 플랫폼 (나라장터 연계)
- **우수조달물품 지정**: 품질·기술 우수 제품을 조달청이 지정 → 종합쇼핑몰에 등록
- **벤처나라**: 벤처·스타트업 제품 전용 구매 채널

**[3.4.5] 사회적 가치/ESG 조달**

- **사회적경제기업 우선구매**: 사회적기업, 마을기업, 협동조합 등 제품 우선 구매
- **장애인기업 우선구매**: 장애인기업 생산품 우선 구매 의무
- **ESG 조달**: 환경(E), 사회(S), 지배구조(G) 요소를 조달 평가기준에 반영
- **이음장터**: 사회적경제기업 전용 구매 플랫폼

### 3.5 [주요항목5] 공공조달법률 이해 ⭐⭐⭐ (예상 9문항, 30%) — 최고 비중

#### 📌 출제기준 세세항목
1. 민법 계약 규정 (계약의 성립·효력·해제·해지)
2. 국가계약법령 핵심 규정 (입찰·낙찰·계약체결·이행·분쟁)
3. 지방계약법령
4. 조달사업법
5. 전자조달법
6. 공기업/준정부기관 계약사무규칙

#### 📝 핵심 정리

**[3.5.1] 국가계약법 - 계약의 원칙 ⭐⭐**

| 원칙 | 법적 근거 | 핵심 내용 |
|---|---|---|
| **계약자유의 원칙** | 국가계약법 제5조 | 대등한 지위에서 당사자 합의로 계약 체결 |
| **호혜의 원칙** | 국가계약법 제5조 | 상호 이익이 되는 계약 체결 |
| **권한남용금지** | 국가계약법 제5조 | 우월적 지위를 이용한 불공정 행위 금지 |
| **청렴의 원칙** | 국가계약법 제5조의2 | 입찰·계약 과정에서 금품·향응 수수 금지 |

**[3.5.2] 추정가격과 예정가격 결정 ⭐⭐⭐ (출제 가능성 최고)**

| 구분 | 정의 | VAT | 용도 |
|---|---|---|---|
| **추정가격** | 물품·공사·용역의 조달에 필요한 예상 가격 | **미포함** | 입찰방법 결정 기준 (일반/제한/수의) |
| **예정가격** | 계약을 체결할 때 계약금액의 기준이 되는 가격 | **포함** | 낙찰자 결정 기준 |

**예정가격 결정 절차 ⭐⭐:**
1. **기초금액** 산정 (거래실례가격, 원가계산, 표준시장단가, 감정가격, 견적가격 등)
2. 기초금액 기준 **±2% 범위** 내에서 **15개 복수예비가격** 작성
3. 입찰자가 15개 중 **4개** 선택
4. 선택된 4개 예비가격의 **산술평균** = **예정가격**

**예정가격 결정기준 (국가계약법 시행령 제9조):**

| 순서 | 기준 | 적용 대상 |
|---|---|---|
| 1순위 | **거래실례가격** | 일반 물품/용역 |
| 2순위 | **원가계산** | 신규 제작/특수 용역 |
| 3순위 | **표준시장단가** | 공사 부문 |
| 4순위 | **감정가격** | 토지·건물 등 |
| 5순위 | **거래실례가격 또는 견적가격** | 기타 |

**[3.5.3] 계약방법 분류 ⭐⭐**

**가. 경쟁 유형별:**

| 방법 | 핵심 내용 |

| 일반경쟁 | 자격요건 충족 모든 업체 참가 (원칙) |
| 제한경쟁 | 실적·기술·자본금 등 참가 제한 |
| 지명경쟁 | 수요기관이 특정 업체 지명 |
| 수의계약 | 경쟁 없이 특정 업체와 직접 계약 |

**나. 공급방법별:**

| 방법 | 핵심 내용 |
|---|---|
| **총액계약** | 전체 물량을 하나의 계약으로 |
| **단가계약** | 단가만 정하고 수량은 수요 발생 시 결정 |
| **제3자단가계약** | 조달청이 단가계약 → 수요기관이 직접 구매 |
| **다수공급자계약(MAS)** | 동일 품목 2인 이상 계약, 수요기관 선택 |

|---|---|

| **개산계약** | 개략 금액 정한 후 사후 정산 |
| **사후원가검토조건부계약** | 계약 후 원가를 검토하여 정산 |

**[3.5.4] 낙찰자 결정제도 ⭐⭐⭐ (출제 가능성 매우 높음)**

| 제도 | 적용 대상 | 핵심 내용 |
|---|---|---|
| **적격심사제도** | 물품·용역·공사 일반 | 가격+이행능력 종합 평가, 일정 점수 이상이면서 최저가 |
| **2단계입찰** | 기술·가격 분리 필요 시 | 1단계(기술) → 2단계(가격) 순차 평가 |
| **종합심사낙찰제** | 공사(일정 규모 이상) | 가격+공사수행능력+사회적 책임 종합 평가 |
| **협상에 의한 계약** | 용역·기술 중심 | 기술제안서 평가 → 우선협상대상자 선정 → 협상 |
| **경쟁적 대화** | 복합·혁신 사업 | 대화를 통해 최적 방안 도출 후 입찰 |
| **종합낙찰제** | 물품 + 소프트웨어 | 가격+성능·품질 종합 평가 |
| **희망수량경쟁입찰** | 일정 수량 필요 시 | 입찰자가 공급 희망수량과 가격 동시 제출 |

**[3.5.5] 민법 계약 규정 (계약의 성립·효력·해제·해지) ⭐⭐**

**가. 계약의 성립:**

| 민법 조문 | 핵심 내용 |
|---|---|
| **제527조** (청약의 구속력) | 계약의 청약은 이를 **철회할 수 없다** |
| **제528조** (승낙기간을 정한 청약) | 기간 내 승낙 통지 못 받으면 효력 상실 |
| **제529조** (승낙기간을 정하지 않은 청약) | **상당한 기간** 내 승낙 통지 필요 |
| **제530조** (연착된 승낙의 효력) | 연착된 승낙은 **새로운 청약**으로 봄 |
| **제531조** (격지자간 계약성립시기) | 격지자 간 계약은 **승낙의 통지를 발송한 때** 성립 |
| **제532조** (의사실현에 의한 계약성립) | 승낙 통지 불필요한 경우, 의사표시로 인정되는 사실이 있을 때 성립 |

- **계약 효력**: 허위서류에 의한 계약 → 취소 또는 해제 가능

**[3.6.4] 부정당제재 처분 법리 ⭐**

- 부정당업자 제재는 **행정처분**(재량행위)으로 행정소송 대상
- 비례의 원칙 적용: 제재 기간이 위반 행위에 비해 과중한 경우 취소 가능
- 제재 기간 감경 사유: 자진 시정, 경미한 위반, 초범 등

**[3.6.5] 국가계약분쟁조정제도 ⭐⭐**

- 2013년 도입
- **6단계 절차**: ①이의신청 → ②회신 → ③재심청구 → ④수리/각하 → ⑤심사·조정안 → ⑥통보
