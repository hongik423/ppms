## 개발 원칙
1. **출제기준 근거 학습**: 모든 토픽 상세 페이지에 출제기준 섹션 표시 (amber 테마)
2. **거꾸로 학습법**: 문제풀이 → 정답확인 → 교재연계 해설 순서
3. **교재 연계**: enhanced-explanations.json의 `textbook` 필드로 동적 색상 적용
4. **TypeScript strict**: `npx tsc --noEmit` 항상 통과 필수
5. **Vercel 자동 배포**: main 브랜치 push → 자동 배포
